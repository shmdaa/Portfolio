HTML Table Basics

How to Define a Table?

A table in HTML is defined using the <table> element.

Using HTML Elements for Tables

<tr>: Defines a table row.

<th>: Defines a table header cell.

<td>: Defines a table data cell.

<thead>: Groups the header content in a table.

<tfoot>: Groups the footer content in a table.

<caption>: Defines a table caption.

Difference Between colspan and rowspan

colspan is an attribute used to specify the number of columns a cell should span horizontally.

rowspan is an attribute used to specify the number of rows a cell should span vertically.

Difference Between Cell Spacing and Padding

cellspacing is an attribute used to control the space between cells in a table.

padding is a CSS property used to control the space between the content of a cell and its border.

Defining Table Borders Using CSS

You can define table borders using CSS by setting the border property on the <table> and related elements.

Controlling Border Widths Using CSS

You can control border widths using the border-width property in CSS. For example: border-width: 2px;.
